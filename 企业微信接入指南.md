# 企业微信接入指南

## 快速开始

### 1. 前置准备

- 企业微信管理后台账号（免费注册企业）
- 企业微信应用（自建应用）

### 2. 创建企业微信应用

1. 登录 https://work.weixin.qq.com
2. 进入 **应用管理** → **自建应用** → 创建新应用
3. 记录以下信息：
   - **CorpID**：企业 ID（我的企业页面底部）
   - **AgentID**：应用 ID
   - **Secret**：应用密钥

### 3. 配置 .env

```bash
# 复制模板
cp server/.env.example server/.env

# 编辑 .env 填入企业微信配置
WECOM_CORP_ID=你的企业ID
WECOM_CORP_SECRET=你的应用密钥
WECOM_AGENT_ID=你的应用ID
WECOM_TOKEN=随机生成的Token
WECOM_ENCODING_AES_KEY=随机生成的43位AES Key
```

### 4. 启动服务

#### 方式一：tmux 一键启动（推荐）

```bash
cd chat_with_llm_local
bash scripts/start-wecom-tmux.sh
```

#### 方式二：两个终端分别运行

终端 1：
```bash
cd chat_with_llm_local
bash scripts/run-wecom-mcp.sh
```

终端 2：
```bash
cd chat_with_llm_local
bash scripts/run-serveo.sh
```

### 5. 设置回调 URL

启动后会输出类似：
```
Forwarding HTTP traffic from https://908e75f07172cc8d-113-249-13-184.serveousercontent.com
```

在企业微信管理后台 → 应用设置：
- 接收消息 → 回调 URL：`https://xxxx.serveousercontent.com/wecom/callback`
- Token 和 EncodingAESKey 与 .env 中一致

### 6. 测试

```bash
# 使用 CLI 模式测试
cd server
python local_client.py --cli

# 测试企业微信工具
> 你: 获取企业微信状态
```

## 工具说明

| 工具 | 说明 |
|------|------|
| `wecom:send_message` | 发送文本消息给用户 |
| `wecom:send_markdown` | 发送 Markdown 消息 |
| `wecom:send_group_message` | 发送群消息 |
| `wecom:get_unread_messages` | 获取未读消息 |
| `wecom:get_status` | 获取连接状态 |

## 工作流程

```
用户在企业微信发消息
    ↓
企业微信服务器 POST 到回调 URL
    ↓
serveo 转发到本地 Flask (端口 9800)
    ↓
消息进入 message_queue
    ↓
LLM 调用 wecom:get_unread_messages 获取消息
    ↓
LLM 处理后调用 wecom:send_message 回复
```

## 内网穿透方案

由于个人电脑没有公网 IP，需要使用内网穿透将本地服务暴露到公网。

### serveo（推荐）

- 基于 SSH 反向隧道
- 无需安装额外工具
- 支持代理环境
- 免费版 URL 每次启动会变化

```bash
bash scripts/run-serveo.sh
```

### ngrok（备选）

- 需要注册账号
- 免费版不支持代理环境
- 付费版支持固定域名

```bash
# 安装
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
tar -xzf ngrok-v3-stable-linux-amd64.tgz
sudo mv ngrok /usr/local/bin/
ngrok config add-authtoken YOUR_TOKEN

# 启动
ngrok http 9800
```

### frp（需要公网服务器）

如果有公网 VPS，可以使用 frp 获得固定域名。

配置文件见 `scripts/frpc.toml.example`。

## 常见问题

**Q: serveo 免费版 URL 每次变化怎么办？**
A: 每次启动后重新在企业微信后台设置回调 URL。或付费 serveo/ngrok 获得固定域名。

**Q: 企业微信个人可以用吗？**
A: 需要注册企业（免费），个人微信无法直接使用。但企业微信可以添加外部联系人（包括个人微信用户）。

**Q: 消息延迟多少？**
A: 通常 1-3 秒，取决于网络状况。

**Q: 支持发送图片/文件吗？**
A: 当前版本支持文本和 Markdown，图片/文件需要额外实现上传接口。
